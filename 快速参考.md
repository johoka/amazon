## 🔧 修改的3个现有文件

### 1. `order.js` - 修复竞态条件

**修改位置**: 第95-107行 + 第483-496行

**修改前** ❌:
```javascript
let setMaxTabs = false;
let maxTabs = 3;
chrome.storage.local.get("maxTabs", function (result) {
  maxTabs = result.maxTabs || 5;
  setMaxTabs = true;
});
while (!setMaxTabs) {
  await sleep(1000);  // 轮询等待！
}
```

**修改后** ✅:
```javascript
const result = await getMaxTabsAsync();
const maxTabs = result;

// 新增函数
function getMaxTabsAsync() {
  return new Promise((resolve, reject) => {
    chrome.storage.local.get('maxTabs', (result) => {
      if (chrome.runtime.lastError) {
        reject(chrome.runtime.lastError);
      } else {
        resolve(result.maxTabs || 5);
      }
    });
  });
}
```

**影响**: ✅ 消除轮询延迟，邀请流程立即启动

### 2. `background.js` - 修复内存泄漏

**修改位置**: 第21-59行

**修改前** ❌:
```javascript
chrome.tabs.create({ url: url, active: request.active }, function (tab) {
  // ... 超时设置 ...

  // 每次都注册新监听器，永不移除！
  chrome.tabs.onUpdated.addListener(function (tabId, changeInfo, updatedTab) {
    if (tabId === tab.id && changeInfo.status === "complete") {
      chrome.tabs.sendMessage(tab.id, { startInvite: true, orderId: orderId });
    }
  });
  // 监听器堆积 -> 内存泄漏
});
```

**修改后** ✅:
```javascript
chrome.tabs.create({ url: url, active: request.active }, function (tab) {
  // ... 超时设置 ...

  // 创建命名监听器，在完成后移除
  const updateListener = function (tabId, changeInfo, updatedTab) {
    if (tabId === tab.id && changeInfo.status === "complete") {
      chrome.tabs.onUpdated.removeListener(updateListener); // ✅ 移除！
      chrome.tabs.sendMessage(tab.id, { startInvite: true, orderId: orderId });
    }
  };

  chrome.tabs.onUpdated.addListener(updateListener);
});
```

**影响**: ✅ 消除内存泄漏，长时间运行内存稳定
